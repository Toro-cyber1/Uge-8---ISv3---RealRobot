# Real Robot - Week 8
Projektet er blevet udvidet, **robot-IP og “dry run”-tilstand styres fra GUI’en**, så app fra uge 7 kan bruges både med:

- URSim (Docker).
- En fysisk UR-robot (ved at skifte IP-adresse).
- Ren “dry run”, hvor der kun logges, hvad der ville blive sendt.

Videoen `Presentation week 8` først kørsel mod URSim (Dry run slået fra, robotten bevæger sig), og bagefter en dry-run hvor der kun logges, hvad der ville blive sendt til en fysisk robot.
